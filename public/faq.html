Foire aux questions (FAQ)

Réponses aux questions les plus fréquentes sur World Puzzle. Pour les règles détaillées de chaque mode, consultez Comment jouer.

Compte et inscription

Faut-il créer un compte ?

Non. Le jeu fonctionne sans inscription. Vous ouvrez le site et jouez directement. Certaines préférences (comme la langue d’interface) peuvent être stockées uniquement sur votre appareil via le navigateur.

Mes parties sont-elles sauvegardées sur un serveur ?

Le jeu ne propose pas de profil cloud : les scores et parties en cours ne sont en général pas synchronisés entre plusieurs appareils. Si vous videz les données du site dans votre navigateur, les réglages locaux peuvent être réinitialisés.

Langues

Le site est-il disponible en anglais ?

L’interface du jeu propose un choix de langue (français / anglais) depuis les réglages du jeu. Les pages d’information que vous lisez ici sont rédigées en français pour détailler le fonctionnement et les engagements du site ; elles complètent l’expérience pour les visiteurs et les outils de modération publicitaire.

Navigateur et matériel

Quels navigateurs sont recommandés ?

Utilisez un navigateur récent (Chrome, Firefox, Safari, Edge). Le jeu s’appuie sur des technologies web modernes (canvas, JavaScript). Les versions très anciennes ou les modes « navigation privée » très restrictifs peuvent limiter le stockage local ou le chargement des cartes.

Puis-je jouer sur téléphone ou tablette ?

L’interface est pensée pour le bureau et la souris (glisser-déposer, molette). Sur écran tactile, le jeu peut être utilisable selon l’appareil, mais l’expérience optimale reste sur ordinateur avec une résolution confortable. Si un geste ne répond pas, essayez un autre navigateur ou mettez à jour le système.

La carte ne charge pas ou reste vide

Vérifiez votre connexion Internet, désactivez temporairement les bloqueurs de contenu agressifs pour ce domaine, puis rechargez la page. Si le problème persiste, essayez un autre navigateur ou un réseau différent (certaines connexions d’entreprise filtrent les ressources).

Publicités et confidentialité

Pourquoi vois-je une bannière sur les cookies ou la publicité ?

Le site peut afficher des publicités (par exemple via Google AdSense). Une bannière vous permet d’accepter ou de refuser le chargement du script publicitaire selon votre choix. Les explications sur les données et les cookies se trouvent sur la page Confidentialité et cookies.

Où est le fichier ads.txt ?

Il est publié à l’adresse /ads.txt à la racine du site ; il sert à identifier l’éditeur autorisé pour certaines annonces programmatiques, conformément aux usages du secteur.

Contenu et cartes

D’où viennent les frontières et les noms ?

Les fonds cartographiques s’appuient sur des données Natural Earth (domaine public). Les choix de simplification et d’affichage sont décrits sur la page À propos. Les frontières politiques peuvent évoluer dans la réalité ; le jeu reflète une photographie des données à la date de mise à jour des fichiers.

Pourquoi certains territoires n’ont pas de mode drapeaux ?

Sur certaines cartes très fragmentées (départements français, États américains), le mode drapeaux peut être indisponible ou limité faute de drapeaux ISO standardisés pour chaque subdivision. Un message dans le jeu vous l’indique lorsque c’est le cas.

Contact

Pour signaler un bug, une erreur géographique sensible ou une question juridique, utilisez les coordonnées ou le canal indiqués sur la page Mentions légales.
